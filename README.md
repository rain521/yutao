# yutao
一个node实例


一、项目需要技术与工具

  前端：html、css、js、jq、ajax
  后端：nodeJs、ejs（html）、MongoDB(数据库)
  
二、项目运行：

  ①确保电脑安装了Node与MongoDB
  ②启动MongoDB
  ③运行命令：npm install (先下载项目所需的第三方模块)
  ④启动项目命令：node app.js
  ⑤打开浏览器输入地址：localhost:5001
